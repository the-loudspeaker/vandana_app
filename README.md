# vandana_app

A Order management app for [Vandana Mobile shoppe](https://maps.app.goo.gl/A6MA6BUwm922xjDN7).

## Getting Started

make sure to add `--dart-define=SUPABASE_KEY=your_supabase_anon_key` to run args.
